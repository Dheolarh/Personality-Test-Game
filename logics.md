at the end of the test, we will display the result to the user. the result will be one of the following:
- Early Achiever
- On the Go Hustler
- Comfort Seeker
- Nostalgic Traditionalist
- Culinary Alchemist

each result will be from the options
each option has a corresponding result for different question as seen in the questions.md file

for example:
- Early Achiever: A
- On the Go Hustler: C
- Comfort Seeker: B
- Nostalgic Traditionalist: D

if a user does the question and most of his answers are options will e.g early achiever, then we display the early achiever result to the user.

in a scenario where a user has equal number of answers for two different results, we will display the result that comes first in the order of the results.

for example:
- if a user has 3 answers as early achiever and 3 answers as on the go hustler, we will display the result that got to 3 first.
